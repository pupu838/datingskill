## 效果示例

### 场景：用户在图书馆看到一个女生

```
用户 ❯ 我在图书馆自习室看到一个女生，怎么搭讪？

搭讪.skill ❯
  图书馆场景属于"校内日常"，参考破圈社交 + 食堂场景的变体。

  第一步：不要直接搭讪。先确认她是不是经常来这个位置，
         下次你早点来坐附近，建立"熟悉的陌生人"感。

  第二步：自然破冰。
         "同学，你经常来这个位置吗？我之前也常坐这儿。"
         或者借东西："同学，有充电宝吗？手机快没电了。"

  第三步：如果她回应友善，学习结束走的时候：
         "我明天也来，帮你占个位？"
         顺势加微信。

  核心要点：
  - 图书馆是"长期场景"，不要一次性梭哈
  - 不要在她学习时打扰，选休息时段
  - "占位"是天然的二次联系理由
```

### 场景：加上微信不知道怎么聊

```
用户 ❯ 昨天小吃街加了一个女生微信，现在不知道怎么开口

搭讪.skill ❯
  你已经有了"场景锚点"——小吃街。不要浪费它。

  推荐开场（三选一）：
  1. "昨天那家烤冷面真的好吃，今天又去了😂"
  2. "你后来还吃了别的吗？我回去拉肚子了💀"
  3. 发一张你昨天在小吃街拍的照片（不要拍到她）

  节奏控制：
  - 她回一句，你回两三句，不要写小作文
  - 聊到开心时主动结束："先不说了，我去上课，回头聊"
  - 聊天目的不是展示自己，是让她多说话

  下一个动作：
  - 两天后再聊，用"顺便"策略约：
    "我刚好要去小吃街，你上次不是说还想吃XX吗？"
```

## 设计理念

### 为什么是"知识库"而非"剧本"？

真实恋爱不是照剧本演的。这个 Skill 提供的是**场景框架 + 核心原则 + 台词参考**，而不是死记硬背的句子。你知道"为什么这样做"之后，自然能在任何场景下做出正确反应。

### 三条核心原则

1. **增大基数**：10 个女生里有 1 个喜欢你，认识 100 个就有 10 个。扩大社交圈比提升话术更重要。
2. **自然接触**：最好的搭讪发生在有"同框理由"的场景。街头拦截不如同行、同排队、同活动。
3. **轻松心态**："谈不到找我补一个"——被拒绝不是你的问题，只是不匹配。
